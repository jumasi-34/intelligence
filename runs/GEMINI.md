# GEMINI.md (runs/ 로컬 가이드라인 및 인덱스)

이 문서는 `intelligence/runs/` (자율 작업 이력 및 아티팩트 보관 레이어) 고유의 로컬 규칙과 파일 정보를 신속히 인지하기 위한 마이크로 가이드라인입니다.

---

## 1. 로컬 핵심 제약 (Local Rules)

* **감사 및 추적성 수호 (Audit Trail Preservation)**: 이 폴더에 축적되는 모든 에이전트 자율 가동 결과(Run ID별 디렉터리, `test.log`, `diff.patch` 등)와 근본 원인 분석 로그(`reverse-sync-prevention.md`)는 시스템의 상태 변화와 자가 결함 진단을 위한 영속적인 감사 데이터(SSOT)입니다. 임의로 소거하거나 변조하지 마십시오.
* **중앙 피드백 연동 (Central RCA Feedback)**: 에이전트 자율 구동 중 발생하는 아키텍처 경계 위반, 린트 에러, 순환 참조 등 중대 결함이 감지 및 복구될 경우, 해당 근본 원인(RCA)과 재발방지책을 반드시 `reverse-sync-prevention.md`에 추가하여 향후 구동되는 에이전트들의 자가 교정 및 참조 지식으로 활용해야 합니다.
* **영구 아티팩트와 휘발성 아카이브 분리**: 개별 실행 로그는 고유 ID를 기반으로 디렉터리화하여 보관하되, 범용적인 동기화 오류 및 구조 위반의 RCA 트랜드는 수치화/이력화하여 관리의 직관성을 보장하십시오.

## 3. 아티팩트 보관 폴더 구조 및 규격 (Artifact Specifications)

각 실행은 고유한 실행 ID 포맷인 `run_YYYYMMDD_HHMMSS_[hash]` 아래에 단독 폴더로 영속 분리 저장됩니다.

```
intelligence/runs/
│
├── run_20260530_185937_c91a3b/     # [실행 예시 디렉터리]
│   ├── task.md                    # 에이전트 실행 대상 피처 및 버그 명세서
│   ├── context-manifest.yaml      # 주입된 도메인 룰 및 코딩 표준 파일 경로 목록
│   ├── plan.md                    # 코딩 수정 가동 전 에이전트가 도출한 실행 계획 및 결정 근거
│   ├── diff.patch                 # 수정한 소스 코드의 로컬 Git Diff 패치 파일
│   ├── test.log                   # 빌드 및 `make verify` 구문 검사 터미널 출력 로그
│   ├── review.md                  # qa-reviewer가 정적 분석을 통해 피드백한 리뷰서
│   └── risk.md                    # ops-release-guard가 측정한 배포 안정성 보고서
│
└── ...
```

### 7대 핵심 아티팩트 세부 규격

1. **`task.md`**: 에이전트가 시작 단계에서 할당받은 작업 목표를 마크다운 형식으로 기록합니다. 대상 파일, 오류 메시지 원본, 작업자 세션 및 도메인 정보가 포함됩니다.
2. **`context-manifest.yaml`**: 에이전트 추론 정확성을 추적하기 위해 당시 로드된 컨텍스트 파일의 체크섬(MD5 Hash)과 물리 파일 경로 목록을 기록합니다.
3. **`plan.md`**: 에이전트가 실제 소스 코드를 변경하기 전, 실행 계획과 결정 근거를 요약 기록합니다.
4. **`diff.patch`**: 에이전트가 완성해 낸 파일 차이점(`git diff` 포맷)을 온전히 저장하여 롤백 및 시니어 승인 용도로 사용합니다.
5. **`test.log`**: 에이전트 기동 직후 `tests/verify_code.py` 및 하네스 테스트를 트리거한 터미널 출력 결과물입니다.
6. **`review.md`**: `qa-reviewer` 에이전트가 `diff.patch`를 전수 스캔하여 결함, 보안 취약점, 테스트 누락 여부를 진단하고 작성한 리뷰 보고서입니다.
7. **`risk.md`**: `ops-release-guard` 에이전트가 이 코드 변경 사항이 프로덕션 배포 시 유발할 수 있는 잠재 리스크를 종합 진단한 보고서입니다.
